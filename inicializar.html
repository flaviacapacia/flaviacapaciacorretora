🚀 Inicializar Projeto

Guia interativo para iniciar os servidores do projeto

🏠
Site Principal
Servidor HTTP para HTML estático
Pré-requisitos
  • Python 3 instalado
🐍 Python
🟢 Node.js
1 Abra o terminal na pasta do projeto
2 Execute o comando:
Terminal
python3 -m http.server 8080
3 Acesse no navegador:
http://localhost:8080
1 Abra o terminal na pasta do projeto
2 Execute o comando:
Terminal
npx http-server -p 8080
3 Acesse no navegador:
http://localhost:8080
ℹ️ Pressione Ctrl+C no terminal para parar o servidor
🤖
Dashboard React
Bot Google Ads com IA
Pré-requisitos
  • Node.js 18+ instalado
  • npm ou yarn
1 Navegue para a pasta do dashboard
Terminal
cd bot-google-ads/dashboard
2 Instale as dependências (primeira vez)
Terminal
npm install
3 Inicie o servidor de desenvolvimento
Terminal
npm run dev
4 Acesse no navegador:
http://localhost:3000
⚠️ A API precisa estar rodando para funcionalidade completa

2 - Dashboard React

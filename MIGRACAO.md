## 🚀 Como Usar o Novo Repositório?

### 1. Clone o Novo Repo
```bash
git clone https://github.com/flaviacapacia/bot-google-ads-ia.git
cd bot-google-ads-ia
```

### 2. Configure o Ambiente
```bash
cp .env.example .env
# Edite .env com suas credenciais
```

### 3. Inicie o Sistema
```bash
# Com Docker
docker-compose up -d

# Ou manualmente
./start.sh
```

### 4. Acesse o Dashboard
```
http://localhost:3000
```

## 📝 Passo a Passo Detalhado

### Método 1: Interface Visual (Mais Fácil)

#### Passo 1: Acesse o GitHub
Abra seu navegador e vá para:
```
https://github.com/flaviacapacia/flaviacapaciacorretora
```

#### Passo 2: Localize o Botão Verde
- Olhe no **canto superior direito** da página
- Procure um botão **VERDE** escrito **"Code"**
- Ele fica ao lado de outros botões como "Star" e "Fork"

#### Passo 3: Clique no Botão
- Clique uma vez no botão verde "Code"
- Um menu pequeno vai aparecer (dropdown)

#### Passo 4: Clique em "Download ZIP"
- No menu que apareceu, role até o final
- A **última opção** é "Download ZIP" com um ícone de seta para baixo (📥)
- Clique em "Download ZIP"

#### Passo 5: Aguarde o Download
- O navegador vai perguntar onde salvar
- Escolha a **Área de Trabalho** ou pasta **Downloads**
- Aguarde o download terminar

---

### Método 2: Link Direto (Mais Rápido)

Se você não quer procurar o botão, use o link direto:

```
https://github.com/flaviacapacia/flaviacapaciacorretora/archive/refs/heads/copilot/create-ads-generation-bot.zip
```

**Como usar:**
1. Copie o link acima
2. Cole no seu navegador
3. Pressione Enter
4. O download começa automaticamente!

---

### Método 3: Via Releases

Outra forma é baixar pela página de releases:

1. Vá para: https://github.com/flaviacapacia/flaviacapaciacorretora/releases
2. Procure pela versão mais recente
3. Em "Assets", clique no arquivo ZIP
4. Download!

## 🖼️ Descrição Visual Detalhada

### Como é o Botão "Code"?

**Características:**
- **Cor:** Verde (#2da44e no GitHub)
- **Localização:** Canto superior direito da página
- **Tamanho:** Botão médio, destaque visual
- **Texto:** Escrito "Code" em branco
- **Ícone:** Pode ter um ícone de código (</>)

### Como é o Menu Dropdown?

**Quando você clica em "Code", aparece:**
- Fundo branco
- Bordas arredondadas
- Sombra sutil
- 3 seções principais:
  1. **Clone** (com URLs)
  2. **Open with GitHub Desktop**
  3. **Download ZIP** ← Esta é a que você quer!

### Como é a Opção "Download ZIP"?

**Características:**
- Está na **parte inferior** do menu
- Texto: "Download ZIP"
- Ícone: Seta para baixo (📥)
- Cor do texto: Cinza escuro
- Ao passar o mouse, fica destacada

## ⚠️ Problemas Comuns e Soluções

### 1. "Não vejo o botão Code"
**Possíveis causas:**
- Você não está na página principal do repositório
- Está vendo um arquivo específico

**Solução:**
- Clique no nome do repositório (no topo) para voltar à página principal
- A URL deve ser: `github.com/flaviacapacia/flaviacapaciacorretora`

### 2. "O menu não abre quando clico"
**Possíveis causas:**
- JavaScript desabilitado
- Bloqueador de anúncios interferindo
- Problema de carregamento

**Solução:**
- Recarregue a página (F5)
- Desative temporariamente o bloqueador
- Tente outro navegador
- Use o link direto do Método 2

### 3. "Download não inicia"
**Possíveis causas:**
- Pop-ups bloqueados
- Configurações de download

**Solução:**
- Permita downloads do GitHub
- Verifique se navegador não bloqueou
- Use o link direto

### 4. "Arquivo muito grande"
**O que fazer:**
- O projeto completo tem alguns MB
- Use WiFi se possível
- Aguarde pacientemente
- Não interrompa o download

### 5. "ZIP corrompido após baixar"
**O que fazer:**
- Baixe novamente
- Use outro navegador
- Verifique espaço em disco
- Desative antivírus temporariamente

### 6. "Preciso de versão específica"
**Solução:**
- Use a página de Releases
- Ou mude de branch antes de baixar
- Ou use Git para clonar

### 7. "Não tenho acesso ao GitHub"
**Alternativas:**
- Peça para alguém baixar e enviar
- Use VPN se GitHub estiver bloqueado
- Tente pelo celular

## 📦 Após Baixar o ZIP

### Arquivo Baixado

Você vai ver um arquivo com nome tipo:
```
flaviacapaciacorretora-copilot-create-ads-generation-bot.zip
```

**Detalhes:**
- **Tamanho:** Alguns MB
- **Tipo:** Arquivo compactado ZIP
- **Localização:** Pasta Downloads (geralmente)

### Como Extrair

**Windows:**
1. Clique com botão direito no arquivo ZIP
2. Escolha "Extrair tudo..." ou "Extract All..."
3. Escolha onde extrair (recomendo Área de Trabalho)
4. Clique em "Extrair"

**Mac:**
1. Dê duplo clique no arquivo ZIP
2. Ele extrai automaticamente

**Linux:**
1. Clique direito → "Extrair aqui"
2. Ou use: `unzip nome-do-arquivo.zip`

### Depois de Extrair

1. Entre na pasta que foi criada
2. Você verá vários arquivos, incluindo:
   - `LAUNCHER.html` ← **Comece por aqui!**
   - `start.html`
   - `index.html`
   - Pasta `bot-google-ads/`
   - E outros arquivos

3. Dê **duplo clique** em `LAUNCHER.html`
4. O arquivo vai abrir no seu navegador
5. Siga as instruções na tela!

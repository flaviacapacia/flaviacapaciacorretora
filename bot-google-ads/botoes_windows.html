📋 Como Usar

  • 1. Baixe esta pasta completa do GitHub
  • 2. Clique duas vezes em INICIAR.bat ou MENU_WINDOWS.bat
  • 3. Escolha a opção desejada no menu
  • 4. Aguarde os servidores iniciarem
  • 5. O navegador abrirá automaticamente!

🔗 URLs do Sistema

  • 📊 Dashboard: http://localhost:3000
  • 🌐 API: http://localhost:3001
  • 🤖 Bot: http://localhost:8000
  • 📚 Bot Docs: http://localhost:8000/docs

💡 Dicas Rápidas

  • ✅ Use MENU_WINDOWS.bat para acesso fácil a todas funcionalidades
  • ✅ iniciar_tudo.bat inicia os 3 componentes de uma vez
  • ✅ abrir_admin.bat abre o painel mesmo se já estiver rodando
  • ✅ parar_tudo.bat encerra todos os processos de uma vez
  • ✅ Configure as chaves de API no menu "Configurações"

⚙️ Requisitos

  • ✔️ Node.js 18+ (Baixar)
  • ✔️ Python 3.8+ (Baixar)
  • ✔️ Windows 10/11
  • ✔️ Chave da API Mistral (configure no .env)

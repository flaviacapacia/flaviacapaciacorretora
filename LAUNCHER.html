📋 Como Usar Este Projeto (3 Passos Simples)

1

📥 Baixe o Projeto

No GitHub, clique no botão verde "Code" e depois em "Download ZIP". Salve na sua área de trabalho ou pasta Documentos.

2

📂 Extraia os Arquivos

Clique com botão direito no arquivo ZIP baixado e escolha "Extrair aqui" ou "Extrair tudo". Uma pasta será criada com todos os arquivos.

3

🌐 Abra no Navegador

Entre na pasta extraída e dê duplo clique no arquivo LAUNCHER.html (este arquivo) ou start.html. Ele abrirá automaticamente no seu navegador!

🎁 Downloads Rápidos

💡 Dica Rápida: Como Abrir Arquivo HTML

🖥️ Se Você Já Tem os Arquivos no Seu PC

✓

Windows

Vá até a pasta onde você baixou o projeto e dê duplo clique em qualquer arquivo .html (LAUNCHER.html, start.html ou index.html)

✓

Mac

Abra o Finder, navegue até a pasta do projeto e dê duplo clique em qualquer arquivo .html

✓

Linux

Abra o gerenciador de arquivos, vá até a pasta do projeto e dê duplo clique ou clique com botão direito → "Abrir com Navegador Web"

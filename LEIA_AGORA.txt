# 🎯 INSTRUÇÕES SUPER SIMPLES - Copiar Scripts Windows

## O QUE FAZER AGORA (3 Passos)

### ✅ PASSO 1: Copiar Arquivos

**Duplo clique em:**
```
atualizar_scripts_windows.bat
```

**Digite quando perguntar:**
```
C:\Users\SeuNome\CAMINHO\PARA\bot-google-ads-ia
```
(Use o caminho real onde está seu repositório no VS Code)

**Resultado:**
- 12 arquivos copiados ✅
- Script oferece executar criar_atalhos.bat
- Diga "S" para SIM!

---

### ✅ PASSO 2: No VS Code

Os arquivos aparecerão no VS Code!

**No terminal do VS Code:**
```bash
git add *.bat *.html GUIA_WINDOWS.md
git commit -m "Add Windows desktop scripts"
git push
```

---
